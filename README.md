# 🐾 Капи-Лояльность (Capi-Loyalty)

### Единый геймифицированный хаб лояльности экосистемы Т-Банка

**Капи-Лояльность** — это инновационное решение проблемы фрагментированности бонусных программ внутри SuperApp. Вместо того чтобы путаться в рублях (Black), милях (All Airlines) и баллах (Платинум), пользователь получает единый эмоциональный счет в **«Мандаринках»**, управляемый дружелюбным ИИ-маскотом — **Капибарой**.

---

## 🌟 Ключевые особенности

*   **🍊 Единый баланс «Мандаринок»**: Автоматическая агрегация всех валют лояльности (1 рубль = 1 миля = 1 балл = 1 мандаринка) для наглядного отображения суммарной выгоды.
*   **🤖 AI-Driven Insights**: Интеграция с LLM (через бэкенд) для анализа истории трат пользователя и выдачи персональных советов прямо в баббле маскота.
*   **🎮 Геймификация (Daily Check-in)**: Интерактивное «Мандариновое дерево», позволяющее собирать ежедневный урожай игровых бонусов.
*   **🎭 Динамическая персонализация**: Весь контент приложения (маскот, офферы партнеров, квесты) автоматически подстраивается под финансовый сегмент пользователя (LOW, MEDIUM, HIGH).
*   **📊 Предиктивная аналитика**: Наглядные графики выплат на основе реальных данных из CSV и математический прогноз накоплений на будущий месяц.
*   **👗 Магазин и Гардероб**: Возможность тратить мандаринки на услуги экосистемы (Т-Мобайл, Страхование) или на кастомизацию своей Капибары.

## 🤖 Интеллектуальное ядро и расширяемость

Одной из ключевых особенностей архитектуры **Capi-Loyalty** является абстракция ИИ-слоя. Мы реализовали паттерн «Провайдер», который делает систему полностью независимой от конкретной LLM.

*   **Multi-Provider Support**: Система поддерживает любых OpenAI-совместимых провайдеров. Вы можете в реальном времени переключать «мозги» Капибары между **Google Gemini**, **OpenRouter**, **SambaNova** или локальными моделями через **Ollama**.
*   **Hot-Swap Configuration**: Смена модели или провайдера происходит через редактирование `providers.json` без необходимости перезагрузки и изменения программного кода бэкенда.
*   **Prompt Engineering**: Системные инструкции вынесены в `prompts.json`, что позволяет гибко настраивать тональность общения (Tone of Voice) и бизнес-логику советов без вмешательства в Python-код.

Это делает решение готовым к масштабированию: при выходе новой, более эффективной модели, интеграция занимает менее 1 минуты.

## 🏗 Архитектура системы

Проект реализован как **двухкомпонентная Fullstack-система**:

1.  **Backend (FastAPI + Pandas)**:
    *   Использует Pandas для обработки данных из 5 CSV-таблиц (имитация банковских БД).
    *   Реализует логику сегментации предложений и агрегации балансов.
    *   Обеспечивает безопасное взаимодействие с ИИ-провайдерами через гибкую систему конфигов.
2.  **Frontend (React + Vite + Tailwind CSS)**:
    *   Высокопроизводительное Mobile-first SPA.
    *   Плавные анимации на базе **Framer Motion**.
    *   Интерактивные графики на **Recharts**.

## 📁 Структура данных (CSV)

Приложение работает на реальных связях между следующими таблицами:
*   `Users.csv` — Профили пользователей.
*   `Accounts.csv` — Счета и балансы.
*   `LoyaltyPrograms.csv` — Типы программ.
*   `LoyaltyHistory.csv` — История начислений (используется для аналитики).
*   `Offers.csv` — Партнерские предложения с привязкой к сегментам.

## 🚀 Быстрый запуск

### 1. Клонирование репозитория
Убедитесь, что у вас установлены **Node.js 18+** и **Python 3.10+**.

```bash
git clone https://github.com/ваш-репозиторий/Capi_Loyality.git
cd Capi_Loyality
```

### 2. Запуск Бэкенда (API)
```bash
cd backend
# Создание виртуального окружения
python3 -m venv venv
source venv/bin/activate  # Для Windows: venv\Scripts\activate
# Установка зависимостей
pip install -r requirements.txt
# Запуск сервера
python main.py
```
*Для работы ИИ: переименуйте `backend/config/providers.json.example` в `providers.json` и добавьте свой API-ключ Gemini.*

### 3. Запуск Фронтенда (UI)
```bash
cd frontend
# Установка зависимостей
npm install
# Запуск в режиме разработки
npm run dev
```
*Приложение будет доступно по адресу: `http://localhost:3000`*
