# 🐾 Капи-Лояльность (Backend) — Единый хаб лояльности Т-Банка

Бэкенд-сервис на базе **FastAPI**, который объединяет фрагментированные программы лояльности (Black, All Airlines, Платинум) в единую геймифицированную экосистему с использованием ИИ-советника.

## 🚀 Основные возможности

*   **Единый баланс «Мандаринок»**: Агрегация кэшбэка, миль и баллов из разных CSV-источников в реальном времени.
*   **Умная сегментация**: Автоматическая фильтрация партнерских офферов на основе финансового статуса пользователя (LOW, MEDIUM, HIGH).
*   **AI-Driven архитектура**: Гибкая интеграция с любыми LLM (Gemini, Qwen, Llama) через OpenAI-совместимый протокол.
*   **Data-Driven конфигурация**: Управление провайдерами ИИ и системными промптами через JSON-конфиги без изменения кода (аналог OpenClaw).
*   **Аналитика и прогноз**: Обработка истории транзакций для построения графиков выгоды.

---

## 🛠 Технологический стек

*   **Язык**: Python 3.10+
*   **Фреймворк**: [FastAPI](https://fastapi.tiangolo.com/) (асинхронный высокопроизводительный API)
*   **Обработка данных**: [Pandas](https://pandas.pydata.org/) (эффективная работа с CSV как с БД)
*   **Валидация**: Pydantic v2
*   **AI SDK**: OpenAI Python Library (Universal Compatibility)
*   **Сервер**: Uvicorn

## 📁 Структура проекта

```text
backend/
├── config/                 # JSON-конфигурации (мозг системы)
│   ├── providers.json      # Настройка AI провайдеров (URL, API-Keys, модели)
│   └── prompts.json        # Хранилище системных промптов и шаблонов

├── models/                 # Pydantic схемы данных для API
├── services/               # Бизнес-логика
│   ├── data_manager.py     # Обработка CSV, Join-ы и расчеты
│   └── ai_service.py       # Универсальный интерфейс взаимодействия с ИИ
├── main.py                 # Точка входа FastAPI и маршрутизация
├── .env                    # Секретные ключи (API Keys)
└── requirements.txt        # Зависимости проекта
```

## ⚙️ Установка и запуск

1. **Клонируйте репозиторий и перейдите в папку бэкенда:**
   ```bash
   cd backend
   ```

2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Запустите сервер:**
   ```bash
   python3 main.py
   ```
   Сервер будет доступен по адресу: `http://127.0.0.1:8000`

## 🤖 Конфигурация ИИ (OpenClaw Style)

Проект поддерживает легкую смену «мозгов» Капибары. Вы можете переключаться между локальными моделями (Ollama) и облачными гигантами.

### Смена провайдера в `config/providers.json`:
```json
{
  "active_provider": "openrouter",
  "active_model": "qwen/qwen-3-next-8b-instruct:free",
  "providers": {
    "openrouter": {
        "baseUrl": "https://openrouter.ai/api/v1",
        "apiKey": "sk-or-v1-..."
    },
    "google": {
        "baseUrl": "https://generativelanguage.googleapis.com/v1beta/openai/",
        "apiKey": "AIzaSy..."
    }
  }
}
```

## 🧠 Логика данных (Бизнес-ценность)

Бэкенд реализует концепцию **«Мандаринового эквивалента»**:
1.  Система находит все счета пользователя в `Accounts.csv`.
2.  Связывает их с типами валют из `LoyaltyPrograms.csv`.
3.  Конвертирует 1 рубль / 1 милю / 1 балл в 1 игровую «Мандаринку» для визуального упрощения выгоды.
4.  Сортирует офферы так, чтобы пользователи сегмента `HIGH` видели премиальные предложения (клининг, инвестиции), а `LOW` — повседневную экономию.
